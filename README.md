# System Resource Monitor

A lightweight Python application for monitoring CPU and RAM usage on Windows, Linux, and macOS.

## Features

- **Real-time Monitoring**: Displays CPU and RAM usage with live updates
- **Visual Indicators**: Progress bars with color coding for easy visualization
- **System Tray**: Runs in the background with quick access from system tray
- **Pause/Resume**: Temporarily pause monitoring to reduce system load
- **Detailed Information**: Shows per-core CPU usage, load average, and detailed RAM statistics
